# 🚗 Next-Gen Intelligent Vehicle Safety System

## 📌 Project Overview

This project is an IoT-based intelligent vehicle system designed to enhance **driver safety and vehicle security** using face recognition, sensors, and real-time monitoring.

---

## 🚀 Features

* 🔐 Face Recognition for secure vehicle access
* 📱 Mobile-based fingerprint unlocking
* 😴 Driver drowsiness detection
* 🍺 Alcohol detection system
* 🚨 Accident detection with GPS & GSM alerts
* 🪑 Seatbelt alert system

---

## 🛠️ Technologies Used

* Python
* Arduino / ESP32
* IoT Sensors
* OpenCV
* MQTT
